Shiny Dashboards

Los dashboards dinámicos nos permiten manipular la data y extraer el conocimiento necesario de forma simple.

He tenido la oportunidad de comparar diferentes herramientas para generar dashboards, y el framework Shiny es, a mi parecer, el mejor.

¿Cómo se construye un dashboard con Shiny?

1. Extraer la data:

  • De un archivo Excel
  • De una base de datos (SQL, Oracle)
  • Desde el API de alguna app (Facebook, Google Analytics, etc)

2. Limiar la data:

Tenemos que limpiar cualquier error en la data (repetición de palabras, espaciados extras, etc, para poder trabajar con ella

3. Darle el formato adecuado a la data para procesarla:

Este es un proceso que se conoce como 'tidy data'. Para conocer más sobre el tema, pueden ver este paper publicado en el Journal of Statistical Software
